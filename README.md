## 💻 Wanna run on your machine?

1. Clone the project

   ```sh
   git clone https://github.com/ironsoul0/samruk-hackathon
   ```

1. Install the dependencies

   ```sh
   cd samruk-hackathon
   npm install
   ```

3. Start the development server

   ```sh
   npm start
   ```
